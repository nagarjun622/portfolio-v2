# Nagarjun Portfolio v2

This is my personal portfolio website built using React.js.
It showcases my projects, skills, and contact information in a clean and responsive design.

## Live Demo

 https://nagarjun-portfolio-v2.netlify.app

## Features

* Modern and responsive UI
* Dark/Light theme toggle
* Project showcase section
* Smooth navigation using React Router
* Landing page with introduction

## Tech Stack

* React.js
* JavaScript
* HTML & CSS
* Netlify (Deployment)
* GitHub (Version Control)

## Project Structure

* `src/pages` → Main pages (Home, Projects, Landing)
* `src/components` → Reusable components (Navbar)
* `public` → Static files

## Installation & Setup

To run this project locally:

```bash
git clone https://github.com/nagarjun622/portfolio-v2.git
cd portfolio-v2
npm install
npm start
```

## Build

```bash
npm run build
```

This portfolio is created to showcase my work and skills for internships and career opportunities.
